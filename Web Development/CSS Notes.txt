* id on css = apply to single
	#css-design{}
* when using container on css always using percentage on width
* margin - top right bot left (top and bot, right and left)
	.box-1{
		margin-top: 5px;
		margin-right: 10px;
		margin-bottom: 5px;
		margin-left: 10px;
	}

	.box-1{
		margin: 5px 10px 5px 10px;
	}
	
	or
	
	.box-1{

* Types of positioning
	- Static - default positon
	- Relative - gumagawa ng sariling position hiwalay sa parent
	- Absolute - top 40px bottom 40px
	- Fixed - kahit anong gawing scroll yun yung position nya
	- Initial - kung ano initail position yun yun (di masyado ginagamit)
	- Inherit - position ng nanay yun yung mamanahin (di masyado ginagamit)
